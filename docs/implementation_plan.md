## 기능 2: 스티커 상세 화면 (테이블 UI)

보상 카드 클릭 → 상세 화면 진입

### UI 구성

```
┌─────────────────────────────────────────────────────────┐
│ ← 뒤로     🍦 아이스크림     5 / 10 ⭐                   │
├─────────────────────────────────────────────────────────┤
│  ┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐     │
│  │     │ 월  │ 화  │ 수  │ 목  │ 금  │ 토  │ 일  │     │
│  ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤     │
│  │ 1주 │ ⭐  │ 🌟  │ 💫  │     │ ⭐  │     │     │     │
│  ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤     │
│  │ 2주 │     │     │     │     │     │     │     │     │
│  └─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘     │
├─────────────────────────────────────────────────────────┤
│  스티커 선택:                                            │
│  ⭐ 🌟 💫 ✨ 💖 🎈 🌈 🦄 🎀 🍀 ...                    │
└─────────────────────────────────────────────────────────┘
```

### 동작 방식
1. 빈 셀 터치 → 선택된 스티커 배치 (애니메이션)
2. 스티커 있는 셀 터치 → 스티커 제거
3. 각 스티커 = 1포인트 (자동 카운트)
